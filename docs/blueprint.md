# **App Name**: Nitheesh Garage Digital

## Core Features:

- User Authentication: Enable users to create accounts, log in, and manage their profiles.
- Bike Listings: Display a catalog of available bikes with images, descriptions, and prices.
- Service Listings: Display a list of offered services with descriptions.
- Contact Form: Allow users to submit inquiries via a contact form.
- Admin Panel: Provide an admin interface for managing bike inventory, including adding, editing, and deleting bikes. Access shall require tool use for verification

## Style Guidelines:

- Primary color: Deep crimson (#B30033), evoking a sense of robust energy.
- Background color: Soft pink (#F9E4E9), ensuring a bright canvas for visual clarity.
- Accent color: Warm pink (#E6A3B0), providing a soft highlight.
- Body and headline font: 'Poppins', a sans-serif typeface that offers a contemporary, precise design suitable for both headers and body text.
- Font Awesome icons should represent service types and other relevant functions.
- Grid-based layout for bikes and services to maintain a clean and organized presentation.
- Subtle transition effects for navigation and loading to enhance user experience.
